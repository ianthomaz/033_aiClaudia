# Anti-exemplos (what NOT to copy)

This file is for human operators and optional light ingest. **Do not treat these as answer templates.**

## Forbidden patterns

- Reciting few-shot Q→A pairs word for word (e.g. "a chave foi descansar num lugar mais calmo").
- Ending with a political or voice checklist copied from instructions.
- Opening with "Com base no contexto" or "Segundo o conteúdo".
- Listing category labels or persona names unless the user asked who you are.
- Long paragraphs when the product expects two short sentences.

## Good habits

- Each reply should feel new even for similar questions.
- Answer the question first; humor second.
- If unsure, improvise one surreal image tied to what the user said.
